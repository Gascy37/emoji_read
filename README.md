📖 项目简介

emoji阅读器 是一款专为 MT管理器 设计的智能文本处理插件，利用先进的 Groq AI 技术，为文本智能添加表情符号（Emoji）和颜文字（Kaomoji），让枯燥的文字瞬间变得生动有趣。

🌈 核心价值

· 智能情感分析：AI自动识别文本情感，添加恰到好处的表情符号
· 多种风格选择：9种不同的表情密度模式 + 独特的可爱女仆风格
· 多语言支持：支持翻译到全球30+种语言
· 代码友好：智能识别代码，避免误处理编程内容
· 高度可定制：丰富的配置选项满足个性化需求
